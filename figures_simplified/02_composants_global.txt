component diagram - the system is divided into 4 modules. Module 1 (Noa) has a React frontend, a FastAPI backend, a LangChain pipeline, and a reservation agent. Module 2 (Support Assistant) has a Streamlit frontend, a FastAPI backend, a LangChain pipeline, and an appointment and incident agent. Module 3 (Emotional Support) is integrated inside Module 2 and has a distress detector and an empathetic response generator. Module 4 (Harassment Classifier) has a Scala/Spark pipeline, a Python experimentation component, and 4 classification strategies. All modules share a Redis database for sessions and cache, and a ChromaDB vector store for semantic search. All modules that need language generation call an external vLLM server on kaisens.fr running Mistral-7B.
