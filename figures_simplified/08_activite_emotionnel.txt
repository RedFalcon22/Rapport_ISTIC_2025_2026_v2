activity diagram - the emotional support module receives a message from the Support Assistant and analyzes it with the LLM using a distress detection prompt. If no distress is detected the module does nothing and returns to the normal pipeline. If distress is detected it identifies the type of signal such as depressive ideation, suicidal ideation, loneliness, anxiety, or anger. If the signal is suicidal ideation it triggers an urgent alert and generates an immediate crisis response. For any other signal it generates an empathetic response matched to the type of distress. In both cases it then guides the user toward the Netethic support resources and logs the event with the type, confidence level, and timestamp without storing the message content.
