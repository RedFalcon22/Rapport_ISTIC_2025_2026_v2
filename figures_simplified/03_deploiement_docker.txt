deployment diagram - everything runs on a developer machine using Docker Compose. There are 7 containers: a Noa React frontend on port 3000, a Noa FastAPI backend on port 8000, a Support Streamlit frontend on port 8501, a Support FastAPI backend on port 8001, a classifier container on port 8003, a Redis database on port 6379, and a ChromaDB vector store on port 8002. A Visitor accesses the Noa frontend, an Internal User accesses the Support frontend, and an Administrator accesses the Noa backend admin panel. The two frontends talk to their respective backends inside Docker. Both backends and the classifier read from Redis and ChromaDB, and all three call an external vLLM server on kaisens.fr for language generation.
